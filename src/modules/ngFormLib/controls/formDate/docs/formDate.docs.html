Form Date FormDate.js

ngFormLib.controls.formDate

The Form Date directive acts like a macro in that it expands into a labelinput and some auxillary HTML elements, but does change the $scope of those elements.

It works this way because the elements are inter-related. For example, when the required attribute is set to true the label element is changed so that a required-indicator-CSS-class is applied to the element.

Plugin dependency

Form Date requires the form controls service module, the field error controller module , the error message container module and the FormDateInputTemplate.html template to be loaded.

Examples

Use form-date to create a complete HTML structure containing a label and input element that reacts to the state of the field.

Live demo

Usage

Add the form-date directive to an element.

Options

Options can be passed via attributes on the directive.

Name Type Default Description
field-hint string Text that is related to the field but is not the field's label. Often used to provide additional information about the field. This string can be a language-resource-key.
