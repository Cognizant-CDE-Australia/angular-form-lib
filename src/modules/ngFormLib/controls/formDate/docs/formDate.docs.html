Form Date FormDate.js

ngFormLib.controls.formDate

Same as the Form Input directive, but with additional date validation. The date validation is DD/MM/YYYY-centric. Feel free to refactor it to be more international-friendly.

Plugin dependency

Form Date requires the FormControlService module, the field error controller module , the error message container module and the FormDateInputTemplate.html template to be loaded. Additionally, if the mgcrea.ngStrap.datepicker module is loaded (as part of AngularStrap, then a calendar control will be displayed. See the Date Picker API for more options.

Examples

Use form-date to create a complete HTML structure containing a label and input element that reacts to the state of the field.

Live demo

{{ctrl.date1}}

Usage

Add the form-date directive to an element.

Options

Options can be passed via attributes on the directive.

Name Type Default Description
ff-min-date string Minimum valid date. A string representation of a date in dd/mm/yyyy format, or the string 'today'
ff-max-date string Maximum valid date. A string representation of a date in dd/mm/yyyy format, or the string 'today'
field-hint string Text that is related to the field but is not the field's label. Often used to provide additional information about the field. This string can be a language-resource-key.
