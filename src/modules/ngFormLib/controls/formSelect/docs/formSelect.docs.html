Form Select FormSelect.js

ngFormLib.controls.formSelect

The Form Select directive acts like a macro in that it expands into a <label><select> and some auxillary HTML elements, but does change the $scope of those elements.

It works this way because the elements are inter-related. For example, when the required attribute is set to true the <label> element is changed so that a required-indicator-CSS-class is applied to the element.

Plugin dependency

FormSelect requires the form controls service module, the field error controller module , the error message container module and the FormSelectTemplate.html template to be loaded.

Examples

Use form-select to create a complete HTML structure containing a label and select element that reacts to the error state of the field.

Live demo

Usage

Add the form-select directive to an element.

Options

Options can be passed via attributes on the directive.

Name Type Default Description
field-hint string Text that is related to the field but is not the field's label. Often used to provide additional information about the field. This string can be a language-resource-key.
placeholder string Text to display when there is no initial value.
template URL See Form Controls Service options The HTML template to use for the control. The default value can be overridden on the element, or globally by setting the default value in the formControlsService
ff-ng-options expression Required - the AngularJS ngOptions directive, which is applied to the select element. Note that the 'track by' syntax is required in this expression for the control to work correctly with the form-reset directive.
