ngFormLib

About the project

Angular Form Policy is a set of native directives that allow you to change how form validation works, plus a set of accessibility-aware input controls that are easy to integrate into your AngularJS 1.2 application.

Designed with Bootstrap CSS styles, Angular Form Policy is designed to make your forms more accessible and easier to change.

Quick Start

Install and manage Angular Form Policy with Bower. 

      
        $ bower install angular-form-lib --save

Load the required javascript libraries (Bootstrap scripts files are not needed).

      
        <script src="//ajax.googleapis.com/ajax/libs/angularjs/1.3.0/angular.min.js"></script>

Inject the ngFormLibmodule into your application.

      
        angular.module('myApp', ['ngFormLib']);

Custom builds

Angular Form Policy provides independently built modules that can be loaded separately

        
          angular.module('myApp', [ 'ngFormLib.policy', 'ngFormLib.controls.formInput']);

Contribute

Build and work on Angular Form Policy with Grunt.

      
        // Serve and watch docs, ideal to develop
        $ grunt dev
        // Continuous integration
        $ grunt test
        // Build Angular Form Policy
        $ grunt build
