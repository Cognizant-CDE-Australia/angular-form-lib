Required Marker RequiredMarker.js

ngFormLib.controls.requiredMarker

Required Marker is a small directive which emits HTML which can be styled to display a symbol/content to indicate that a form-field is required. The default template makes the marker accessible by hiding the element from a screen reader and reading out 'required' instead.

Plugin dependency

Required Marker requires the form controls service module template to be loaded.

Examples

Use required-marker to add an indicator inside a label element.

Live demo

Usage

Add the required-marker directive to an element.

Options

Options can be passed via attributes on the directive.

Name Type Default Description
hide expression false A scope-expression which causes the required marker to appear (false) or disappear (true)
template URL ngFormLib/controls/requiredMarker/template/RequiredMarkerTemplate.html The HTML template to use for the control. The default value can be overridden on the element, or globally by setting the default value in the formControlsService
