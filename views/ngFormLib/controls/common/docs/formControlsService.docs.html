Form Controls Service FormControlsService.js

ngFormLib.controls.common

The Form Controls Service service provides common functions used by the form controls, and allows default values for the form controls to be changed from a central place.

Plugin dependency

The Form Controls Service requires the RequiredMarker module and the ngFormLib.common.utility module to be loaded.

Options

Options can be configured like so:

angular.module('myApp', ['ngFormLib'])
  .config(['FormControlsServiceProvider', function(FormControlsServiceProvider) {
      FormControlsServiceProvider.defaults.select.template = 'path/to/my/template.html'
  }])
Name Type Default Description
idPrefix String fpFld The prefix string for generated form-control id values. Generated id values are used when the form-control does not have it's own cid attribute with a non-empty value
formCheckbox.template URL ngFormLib/controls/formCheckbox/ template/FormCheckboxTemplate.html Template for the Form Checkbox control
formDate.template URL ngFormLib/controls/formDate/ template/FormDateTemplate.html Template for the Form Date control
formInput.template String <div class="form-group"><label class="control-label"></label><div class="control-row"><input #type# class="form-control"><span ng-transclude></span></div></div> Template for the Form Input control.
Note: The #type# string is replaced with the input's type (text|tel|number|...) prior to the template being compiled. This is due to the type attribute being read-only on an input element (see here)
formRadioButton.template URL ngFormLib/controls/formRadioButton/ template/FormRadioButtonTemplate.html Template for the Form RadioButton control
formSelect.template URL ngFormLib/controls/formSelect/ template/FormSelectTemplate.html Template for the Form Select control
requiredMarker.template URL ngFormLib/controls/requiredMarker/ template/RequiredMarker.html Template for the Required Marker control
