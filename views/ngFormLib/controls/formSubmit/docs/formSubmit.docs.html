Form Submit FormSubmit.js

ngFormLib.controls.formSubmit

Form Submit is a directive which can contain an expression to execute when the form is valid.
This directive is useful when you need to execute a function on the $scope once the form data has been successfully validated. The function would typically do some minor data-transformation on (a copy of) the form's data before sending the data to a service for further processing.

Examples

Add form-submit directive to any form element or to any button inside a form.

Live demo

Usage

Add the form-submit directive to a form or button element, with a value that is a scope-expression.
