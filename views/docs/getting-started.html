ngFormLib

About the project

Angular Form Library is a set of Angular components that allow you to configure how form validation works across your entire site, plus a set of accessibility-aware input controls that are easy to integrate into your AngularJS 1.2+ application.

Designed with Bootstrap CSS styles and markup in mind, Angular Form Library can be easily customised to work with your existing CSS and form markup.

Quick Start

Install and manage Angular Form Library with Bower. 

      
        $ bower install angular-form-lib --save

Load the required JavaScript libraries (Bootstrap's JavaScript files are not needed).

      
        <script src="//ajax.googleapis.com/ajax/libs/angularjs/1.3.9/angular.min.js"></script>

Inject the ngFormLibmodule into your application.

      
        angular.module('myApp', ['ngFormLib']);

Typically you would load both the module and the default policies into your application:

      
        angular.module('myApp', [
          'ngFormLib',
          'duScroll',   // Scrolling behaviour for when the state changes to an error and we wish to focus the field
          'ngFormLib.policy.behaviourOnStateChange',
          'ngFormLib.policy.checkForStateChanges',
          'ngFormLib.policy.stateDefinitions'
        ]);

Custom builds

Angular Form Library provides independently built modules that can be loaded separately:

        
          angular.module('myApp', [ 'ngFormLib.policy', 'ngFormLib.controls.formInput']);

Contribute

Build and work on Angular Form Library with Grunt.

      
        // Serve and watch docs, ideal to develop
        $ grunt dev
        // Continuous integration
        $ grunt test
        // Build Angular Form Library
        $ grunt build
